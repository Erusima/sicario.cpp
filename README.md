# SicarioCPP

## What is it?

SicarioCPP is a simple Sicario C&C client made by Erusima and Dante383 (creator of original Sicario software).

## Compatibility

- [X] Maintaining connection
- [X] Sending packets
- [X] Receiving packets
- [ ] Commands
	- [ ] Execute
	- [ ] Set Interval
	- [ ] Get Architecture
	- [ ] Get System
	- [ ] Get Interval
